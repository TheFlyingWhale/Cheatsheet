What is CSS?

CSS is what allows us to style the structure of HTML

Fonts

Fonts(typefaces) are used to style text.

Ways to import fonts to a project

There are two ways of importing fonts into a project.

Linking fonts in head

You can include fonts by linking them from an external sources in the head element

        
    <head>
        <link href="https://fonts.googleapis.com/css2?family=Heebo:wght@300;400;500;700&display=swap" rel="stylesheet">
    <head>
Including local fonts

In the stylesheet you can set font styles and link them to local files

        
    @font-face{
        font-family: Heebo;
        font-style: normal;
        font-weight: 400;
        src: url(folder/font.ttf) format("truetype");
    }

Box Model

HTML elements are rendered as a box. With CSS we can manipulate how this box behaves on a site

The Box Model consists of these elements:

  • Content(Height & Width)
  • Padding(Space withing the box)
  • Border(Dividing the outside from the inside)
  • Margin(Space outside the box)

Reset box-model

By default a box with the width of 200px and border with 2px width will render a box of 204px

To avoid this we can reset the entire box model and specify a new one: border-box

This code results in the width being 322px (default)
        
    * {
        box-sizing: content-box; (default value)
    }

    h1 {
        border: 1px solid black;
        height: 200px;
        width: 300px;
        padding: 10px;
        }
This code results in the width being 300px
        
    * {
        box-sizing: border-box;
    }

    h1 {
        border: 1px solid black;
        height: 200px;
        width: 300px;
        padding: 10px;
        }

Flow

Flow refers to how elements are being rendered. By default its left to right, top to bottom, in the same order as they exist in the document.

Position

There are seven types of position an element can have:

  • static (default)
  • relative
  • absolute
  • fixed
  • sticky
  • initial
  • inherit
Static

Default render.

Relative

Allows you to position the element relative to its default position.

Combine relative with top bottom left and right to move the element to an desired location.

Absolute

Makes all other elements ignore given element and act like it's not present on the page.

Fixed

Works like absolute in the sence that other elements will ignore it, but the element will be fixed to a position in the browser.

You can scroll and the element will stay on the same location.

Display

Display dictates if an element can share horizontal space with other elements.

The list of available display properties include:

  • inline
  • block
  • inline-block
Inline

Inline elments have a box wraped tightly around its content, only taking up the space necessary to display their content and not requiring a new line after the each element.

Heigth and width can not be set with CSS on inline elements.

You can use inline on elements thats not inline by default to make them appear on the same line.

Block

Block elements are not displayed on the same line as other elements.

By default its width will fill the entire screen, but width can be set.

Unless otherwise specified, they are the height necessary to accommodate their content

Inline-Block

Inline-block display combines features of both inline and block elements.

Inline-block elements can appear next to each other and we can specify their dimensions using the width and height properties.

Clear

The clear property specifies how elements should behave when they bump into each other on the page

  • left
  • right
  • both
  • none
Left

The left side of the element will not touch any other element within the same containing element.

Right

The right side of the element will not touch any other element within the same containing element.

Both

Neither side of the element will touch any other element within the same containing element.

none

The element can touch either side.
