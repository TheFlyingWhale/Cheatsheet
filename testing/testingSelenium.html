What is Selenium?

Selenium is an umbrella project for a range of tools and libraries that enable and support the automation of web browsers

It provides extensions to emulate users interaction with browsers, a distribution server for scaling browser allocation, and the infrastructure of the W3C WebDriver specification that lets you write interchangeable code for all major web browsers.
