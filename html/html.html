Links

Links are what enables a user of a website to browse trough different pages on a site.

Or links can be used to go to other sites.

<a href="url">link test</a>

Attributes:

  • href (url to source)
  • target (how to open source)
  • alt (alternative text)
  • title (extra infromation)
Target

There are four types of target attributes for a link:

  • _self (Default. Opens the document in the same window/tab as it was clicked)
  • _blank (Opens the document in a new window or tab)
  • _parent (Opens the document in the parent frame)
  • _top (Opens the document in the full body of the window)

Lists

HTML lists allow web developers to group a set of related items in lists.

List types

There are three types of lists in HTML:

  • Unordered lists (Like this)
  • Ordered lists
  • Description lists
Ordered list
  1. Wake up
  2. Brush Teeth
  3. Check mail
  4. Do some work
  5. Eat breakfast
        
    <ol>
        <li>Wake up</li>
        <li>Brush Teeth</li>
        <li>Check mail</li>
        <li>Do some work</li>
        <li>Eat breakfast</li>
    </ol>
