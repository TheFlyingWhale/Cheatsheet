What is HTML?

HTML is the structure of a site.

Semantics

Semantics is important for accessibility and SEO.
It allows us to structure the site in a way thats understandable for both page-readers and developers.

Semantic elements inlcudes:

  • Header
  • Nav
  • Main
  • Article
  • Section
  • Aside
  • Details
  • Figure
  • Figcaption
  • Mark
  • Summary
  • Time
  • Footer

Lists

HTML lists allow web developers to group a set of related items in lists.

List types

There are three types of lists in HTML:

  • Unordered lists (Like this)
  • Ordered lists
  • Description lists
Ordered list
  1. Wake up
  2. Brush Teeth
  3. Check mail
  4. Do some work
  5. Eat breakfast
        
    <ol>
        <li>Wake up</li>
        <li>Brush Teeth</li>
        <li>Check mail</li>
        <li>Do some work</li>
        <li>Eat breakfast</li>
    </ol>
Unordered list
  • Milk
  • Sugar
  • Bread
  • Toothpaste
        
    <ul>
        <li>Milk</li>
        <li>Sugar</li>
        <li>Bread</li>
        <li>Toothpaste</li>
    </ul>
Description lists
Coffee
- black hot drink
Milk
- white cold drink
        
    <dl>
        <dt>Coffee</dt>
        <dd>- black hot drink</dd>
        <dt>Milk</dt>
        <dd>- white cold drink</dd>
    </dl>
